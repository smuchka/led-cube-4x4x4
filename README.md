# LedCube 4x4x4
Led cube 4x4x4 on Arduino

![Version 1.0](./led-cube.gif)

# Cube modes
Const | ID | Description
-----|--|------------
CUBE_COLUMN_CIRCLE | 1 | One column running on outside round
CUBE_LIGHT | 2 | Alight cube
CUBE_BLINK | 3 | Blink cube
CUBE_RAIN | 4 | effect "rain"
